The uv package manager

The uv utility (written in Rust, pronounced "you vee") is a popular package manager that is much faster than the default python package manager, pip.

To compare the two, let's run the program sunpyPrintVersion.py, which simply prints the version of sunpy, under both package managers.

Using pip we set up a virtual environment, enter it, then install the sunpy package, like so : 


 $ cd pip/
 $ python -m venv $HOME/sofwareTalk/02_uv/pip/pyEnv      # Make the env
 $ source $HOME/sofwareTalk/02_uv/pip/pyEnv/bin/activate # Enter the env (the command line prompt changes)
 (pyEnv) $ pip install --upgrade pip                     # Update pip in the env (not essential)
 (pyEnv) $ pip install "sunpy[all]"                      # ...and install sunpy
That installs sunpy, with the install taking about 30 seconds on my machine.

Then, still in the env, run the program : 


 (pyEnv) $ ../sunpyPrintVersion.py
 5.1.5
