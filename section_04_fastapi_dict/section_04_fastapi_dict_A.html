Quick review before we FINALLY do FastAPI

In Python we've covered :

  • Dictionaries
  • Lists of dictionaries
  • The Java Script Object Notation (JSON) format for representing these, like so :
    {'approximatePI': 3.1415927, 'unity': 1, 'name': 'Niles Oien', 'ravesOn': true}
    (Aside : JSON booleans are true/false, Python booleans are True/False)
  • Decorators (ie function wrappers indicated by the @ sign)

We've also covered the following utilities :

  • bat - Colorful replacement for "cat"
  • uv -Fast Rust-based package manager
  • ruff - Rust-based python code checker

OK, it's time to look at FastAPI!
