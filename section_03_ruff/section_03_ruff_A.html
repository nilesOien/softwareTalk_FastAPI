The ruff python code check utility

The ruff utility (also written in Rust) is a popular python tool for checking python code.

To install it, add "ruff" to the list of packages to be installed in the file requirements.in and then : 


 $ uv init --name ruff_example --description "An example of ruff" --bare .
 $ uv add -r requirements.in
The script installPackages.sh does the same thing.

Then ruff can be used to check all python files : 


 $ uv run ruff check
 F401 [*] `os` imported but unused
   --> helloWorld.py:8:8
    |
  6 | # $ uv run ruff check --fix
  7 |
  8 | import os
    |        ^^
  9 |
 10 | print("Hello world")
    |
 help: Remove unused import: `os`

 Found 1 error.
 [*] 1 fixable with the `--fix` option.

ruff has detected an unused import in the file helloWorld.py which could be fixed by running 


 $ uv run ruff check --fix

NOTE : If ruff finds any issues, then it will exit with a non-zero exit code. Otherwise if things are fine then the exit code is zero.

That may seem trivial, but it's actually an important point that we'll come back to later.
