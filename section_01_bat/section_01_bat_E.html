Aside : to the bat utility!

The bat utility (written in Rust) is a drop in replacement for cat. It has a built-in pager, so use it like this : 

 $ # Do this :
 $ bat dictionary.py
Not like this : 
 $ # Do not do this!
 $ bat dictionary.py | less
It ties in with git if you're in a repo, so that :
  • Lines that have been added are marked with a +
  • Lines that have been removed are marked with a -
  • Lines that have been modified are marked with a ~
These marks appear in the margin.

See the file youCanUseBat.sh
